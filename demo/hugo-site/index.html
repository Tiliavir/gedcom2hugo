Welcome to the Smith Family Tree demonstration site! This site showcases how gedcom2hugo converts GEDCOM genealogy data into a beautiful Hugo static site.

About This Demo

This demo contains a sample family tree with:

  • 6 individuals across 2 generations
  • 2 family units
  • Life events (births, deaths, marriages)
  • Occupations and locations

How to View

You can view this demo in several ways:

  1. With Hugo: Run hugo server in the demo/hugo-site directory
  2. Static files: The person pages above will load the JSON data and display it dynamically
  3. Direct API access: View the JSON files in /api/individual/ and /api/family/

What Gets Generated

When you run gedcom2hugo, it creates:

  • Markdown content files for each person and family
  • JSON API files with structured data
  • HTML pages that dynamically load and display the genealogical data
